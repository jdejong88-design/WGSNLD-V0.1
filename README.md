# WatergeusLLM

WatergeusLLM is een Nano-GPT model dat specifiek getraind wordt op de Nederlandse taal. 
Het project is momenteel in de trainingsfase met een dataset van ~68 miljoen tokens.

## Architectuur
- **Model:** Nano-GPT (Transformer)
- **Parameters:** ~51.3M
- **Lagen:** 8
- **Embedding Dim:** 512

## Training
De training vindt lokaal plaats op een **NVIDIA A100 (Google Colab Pro) deze is echter nog niet actief** 
en een lokale **GTX 1080**.
