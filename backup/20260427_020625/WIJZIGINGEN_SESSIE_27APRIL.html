4️⃣ 04_train_model.py (Nieuw - Stap 4)

✅ Trainingsscript Gecreëerd

Doel: Nano-GPT model trainen op 50k Nederlandse Wikipedia artikelen

Functionaliteiten:

  • TokenizedDataset klasse: Laadt tokenized sequenties uit bestanden
  • train_epoch functie: Trainen met cross-entropy loss en gradient clipping
  • validate functie: Validatie loop met held-out data
  • Optimizer: AdamW met learning rate scheduling
  • Early stopping: Stop indien validatie loss niet verbetert (patience=2)
  • Checkpointing: Best model automatisch opgeslagen
  • Training history: Loss tracking in JSON voor analyse

Configuratie:

Learning rate: 0.001 Batch size: 32 Epochs: 3 Sequence length: 512 (matching model architecture) Train/Val split: 90/10 Gradient clip: 1.0

Output bestanden:

  • checkpoints/best_model.pt — Best model checkpoint
  • training_history.json — Loss curve data
