🔧 Stap-voor-Stap Proces

Stap 1: Hugging Face Voorbereiding

Eerst moet je een Hugging Face account hebben met een API token.

  • Account aanmaken op https://huggingface.co
  • Email verifiëren
  • Access Token genereren (Settings → Access Tokens)
  • Token opslaan (begint met hf_)

Stap 2: Python Environment Setup

Installeer benodigde Python libraries.

pip install datasets transformers huggingface-hub torch tqdm jupyter

Wat dit doet:

  • datasets - Hugging Face dataset library
  • transformers - Pre-trained models
  • huggingface-hub - API client
  • torch - Deep learning framework
  • jupyter - Interactive notebooks

Stap 3: Directory Structuur Maken

Maak de mappenstructuur voor datasets.

mkdir -p E:\dutch-llm\data mkdir -p E:\Claude\workflow\WatergeusLLM\datasets

Folder layout:

E:\ ├── dutch-llm\data\ (Scripts hier) │ └── download_*.py │ └── Claude\workflow\ └── WatergeusLLM\ └── datasets\ (Data hier) ├── wikipedia_nl\ ├── mc4_nl\ └── squad_nl\

Stap 4: Download Script Maken

Maak een Python script dat datasets downloadt.

cat > E:\dutch-llm\data\download_datasets.py << 'EOF' from datasets import load_dataset from pathlib import Path import shutil TARGET_DIR = Path("E:/Claude/workflow/WatergeusLLM/datasets") datasets = [ ("allenai/c4", "nl", "mc4_nl"), ("wikipedia", "20231101.nl", "wikipedia_nl"), ("squad_v2", None, "squad_nl"), ] for repo, config, name in datasets: print(f"[*] Loading {name}...") try: if config: ds = load_dataset(repo, config, split="train") else: ds = load_dataset(repo, split="train") target = TARGET_DIR / name if target.exists(): shutil.rmtree(target) ds.save_to_disk(str(target)) print(f"[OK] {name} saved!") except Exception as e: print(f"[ERROR] {name}: {e}") EOF

Stap 5: Hugging Face Authenticatie

Log in met je API token.

export HF_TOKEN="hf_your_token_here"

Of via CLI:

huggingface-cli login # Plak je token en druk op Enter

Stap 6: Download Script Uitvoeren

Voer het download script uit.

cd E:\dutch-llm\data python download_datasets.py

Dit zal:

  • Wikipedia NL laden (2.1M artikelen)
  • MC4 Dutch laden (365K samples)
  • SQuAD NL laden (130K Q&A)
  • Alles opslaan naar datasets/
⏱️ Dit duurt 30-60 minuten afhankelijk van internetsnelheid.

💻 Volledige Python Script

download_datasets_complete.py

#!/usr/bin/env python3 """Download Nederlandse datasets van Hugging Face.""" import shutil from pathlib import Path from datasets import load_dataset from tqdm import tqdm # Directories TARGET_DIR = Path("E:/Claude/workflow/WatergeusLLM/datasets") TARGET_DIR.mkdir(parents=True, exist_ok=True) # Datasets to download DATASETS = [ { "name": "Wikipedia NL", "repo": "wikipedia", "config": "20231101.nl", "output": "wikipedia_nl", "split": "train", }, { "name": "MC4 Dutch", "repo": "allenai/c4", "config": "nl", "output": "mc4_nl", "split": "train", }, { "name": "SQuAD NL", "repo": "squad_v2", "config": None, "output": "squad_nl", "split": "train", }, ] def download_dataset(dataset_info): """Download een dataset.""" name = dataset_info["name"] repo = dataset_info["repo"] config = dataset_info["config"] output = dataset_info["output"] split = dataset_info["split"] print(f"\n[*] Downloading {name}...") print(f" Repo: {repo}") try: # Load dataset if config: ds = load_dataset(repo, config, split=split) else: ds = load_dataset(repo, split=split) print(f"[OK] Loaded {len(ds):,} samples") # Save to disk output_path = TARGET_DIR / output if output_path.exists(): shutil.rmtree(output_path) print(f"[*] Saving to {output_path}...") ds.save_to_disk(str(output_path)) print(f"[OK] Saved successfully!") return True except Exception as e: print(f"[ERROR] {name}: {str(e)[:100]}") return False def main(): """Download all datasets.""" print("=" * 60) print("[*] Dutch Datasets Download") print("=" * 60) print(f"Target: {TARGET_DIR}\n") results = {} for dataset_info in DATASETS: success = download_dataset(dataset_info) results[dataset_info["name"]] = success print("\n" + "=" * 60) print("[SUMMARY]") print("=" * 60) for name, success in results.items(): status = "[OK]" if success else "[FAILED]" print(f"{status} {name}") if __name__ == "__main__": main()

❓ Troubleshooting

Probleem: "Directory not found"

Oplossing: Zorg dat je in de juiste map bent.

cd E:\dutch-llm\data # of cd E:\Claude\workflow\WatergeusLLM\datasets

Probleem: "Authentication required"

Oplossing: Zet je HF_TOKEN environment variable.

export HF_TOKEN="hf_your_token_here"

Probleem: Niet genoeg schijfruimte

Oplossing: Zorg voor ~10 GB vrije ruimte.

powershell -Command "Get-PSDrive E"

Probleem: Slow download

Oplossing: Controleer internet, probeer buiten piekuren.
