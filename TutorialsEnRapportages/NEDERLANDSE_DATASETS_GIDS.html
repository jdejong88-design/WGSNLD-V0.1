🔐 Stap 1: Hugging Face Account Setup

Account maken (als je die nog niet hebt)

Ga naar https://huggingface.co en maak een account aan.

Verifieer je email-adres.

Access Token genereren

  1. Log in op HuggingFace
  2. Ga naar Instellingen → Access Tokens
  3. Klik op "New token"
  4. Kies "Read" permission
  5. Kopieer je token (begint met hf_)

🔧 Python Script Uitleg

Hoe het script werkt:

  1. Import libraries: Laden van datasets en bestandsoperaties
  2. Definieer datasets: Lijst van repo IDs en configs
  3. Loop over datasets: Voor elk dataset:
    • Load van Hugging Face
    • Opslaan naar disk
    • Error handling voor gated datasets
  4. Rapportage: Toon status en locaties

🐛 Troubleshooting

Probleem: "Authentication required"

Oplossing:

huggingface-cli login # Plak je token en druk op Enter

Probleem: "Dataset is gated"

Oplossing: Accepteer het dataset op de Hugging Face website (zie Stap 2)

Probleem: "No space left on device"

Oplossing: Zorg voor ~30 GB vrije ruimte op je E:-schijf

powershell -Command "Get-PSDrive E"

Probleem: Download is traag

Tips:

  • Controleer je internetverbinding
  • Download buiten piekuren
  • Sluit andere downloads
