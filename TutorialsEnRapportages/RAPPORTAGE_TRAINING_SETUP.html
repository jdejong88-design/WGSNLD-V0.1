⚙️ Training Configuratie

Hyperparameters

De standaard training configuratie:

Parameter Waarde Beschrijving
Learning Rate 5e-5 (0.00005) Pas voor fine-tuning van pre-trained modellen
Batch Size 8 Samples per GPU per training step
Epochs 3 Aantal passes over volledige dataset
Warmup Steps 100 Graduale LR toename aan start training
Weight Decay 0.01 L2 regularisatie voor overfitting prevention
Save Steps 100 Checkpoint frequency
Eval Steps 100 Validation frequency

Optimizer & Scheduler

Hugging Face Trainer gebruikt standaard:

  • Optimizer: AdamW (Adam met weight decay)
  • Learning Rate Scheduler: Linear warmup dan linear decay
  • Warmup Strategy: 100 stappen lineaire toename naar base LR
  • Gradient Accumulation: Kan worden ingesteld via gradient_accumulation_steps

📊 Data Split & Collation

Train/Eval Split

De tokenized dataset wordt opgesplitst in:

Set Percentage Doelstelling
Training 90% Model parameters aanpassen
Validation 10% Performance monitoring, best model selection

Data Collator

TextDataCollatorForLanguageModeling voorbereiding:

  • Groeperen van samples in batches
  • Padding tot max sequence length
  • mlm=False → Causal Language Modeling (volgende token voorspelling)
  • Attention masks voor proper padding handling

1. Model en Tokenizer Laden

trainer, training_args = trainer_obj.setup_training( dataset=tokenized_dataset, output_dir='./checkpoints', learning_rate=5e-5, batch_size=8, num_epochs=3, warmup_steps=100, weight_decay=0.01 ) # Creëert: # - TrainingArguments met optimale settings # - Trainer met model, data, callbacks # - Train/eval datasets

⚠️ Belangrijke Opmerkingen

GPU/CPU Hardware

Het script detecteert automatisch GPU (CUDA) beschikbaarheid. Op CPU zal training veel langzamer verlopen (~50-100x slower). Voor productie training wordt NVIDIA GPU (RTX 3090, A100, etc.) sterk aanbevolen.

Memory Requirements

GPT-2 base model vereist minimaal 8GB RAM voor batch_size=8. Bij OOM errors:

  • Batch size reduceren naar 4 of 2
  • Gradient accumulation steps gebruiken
  • Model quantization overwegen

Hyperparameter Tuning

De huidige parameters zijn baseline settings. Voor betere resultaten:

  • Learning rate: Test 1e-5, 3e-5, 5e-5, 1e-4
  • Warmup steps: Try 500, 1000 voor grotere datasets
  • Epochs: Meer epochs voor meer data diversity learning

Model Evaluatie

Eval loss wordt automatisch getracked. Na training checken:

  • Eval loss convergence curve in Trainer logs
  • Model generation quality met sample prompts
  • Perplexity op held-out test set
